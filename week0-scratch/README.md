## Week 0 Project

**Road & Roll**

I made a Scratch game, for my CS50 project. In the game I control a baseball with the arrow keys. My goal is to avoid falling monkeys and survive until the timer hits zero.

### Features

- I move the player with the arrow keys.

- Monkeys fall from above as obstacles.

- The game detects when I hit a monkey.

- I earn points as I survive.

- A countdown timer shows how much time is left.

- When the game ends I see a Game Over screen.

- If I survive until the end I get a Victory message.
